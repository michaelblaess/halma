Sternhalma

Sternhalma gegen die KI — schnell, taktisch, offline spielbar. Drei Schwierigkeitsstufen, sieben Themes, Hintergrundmusik und Highscores.

GitHub Repository
Features

Was es kann

Ein vollwertiges Sternhalma-Spiel im Browser — gebaut mit modernem Web-Stack.

🤖

KI-Gegner

Drei Schwierigkeitsstufen mit Minimax-Algorithmus im Web Worker — die KI blockiert nicht den UI-Thread.

🎨

7 Themes

Von Standard-Dark ueber Holz-Optik bis High-Contrast — mit automatischer Hell/Dunkel-Erkennung.

🎵

Sound & Musik

Generierte Sound-Effekte via Web Audio API und lizenzierte Hintergrundmusik mit Shuffle-Playlist.

📱

Offline & PWA

Installierbar als App, offline spielbar dank Service Worker. Responsive von Mobile bis Desktop.

⚡

Speed-Timer

Zeitmessung mit Zehntel-Sekunden, lokale Highscores pro Schwierigkeitsstufe und kopierbares Markdown-Snippet.

♿

Barrierefrei

ARIA-Labels, Keyboard-Navigation, Focus-Styles, Screen-Reader-Support und prefers-reduced-motion.

Tech Stack

Gebaut mit

Moderner Web-Stack ohne Backend — alles laeuft im Browser.

React 19
TypeScript
Vite 7
Web Workers
PWA + Offline

Disclaimer

Halma ist ein klassisches Brettspiel, erfunden 1883 von George Howard Monks. Die Sternhalma-Variante (auch bekannt als "Chinese Checkers") entstand 1892. Diese Webseite ist ein nicht-kommerzielles Hobby-Projekt und steht in keiner Verbindung zu kommerziellen Halma-Produkten oder deren Herstellern.
